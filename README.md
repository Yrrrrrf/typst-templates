This repository contains a collection of customizable Typst templates designed for various document types. These templates provide consistent formatting, university branding support, and professional styling for academic papers, reports, and other document types that I use across all my projects.

## üö¶ Getting Started

To use these templates:

1. Clone or copy the desired template files to your project directory
2. Modify the `.typ` files with your content
3. Run `typst compile main.typ` to generate your PDF

## üìÅ Structure

- `main.typ` - Main document template with university branding
- `/assets` - Images and attribution files
- `/cover` - Cover page templates for various document types
- `/examples` - Complete and simple example implementations
